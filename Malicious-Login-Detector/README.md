# Malicious Login Detector

## Description

Malicious Login Detector is a cybersecurity project written in C that detects suspicious login attempts using classical data structures and algorithms.

## Features

- Load login records from a CSV file
- Detect suspicious users using a Stack
- Simulate login requests using a Queue
- Search using Linear Search and Binary Search
- Sort users using:
  - Bubble Sort
  - Insertion Sort
  - Quick Sort
  - Merge Sort
- Classify users with a Binary Search Tree (BST)
- Analyze attack propagation using Graphs and BFS
